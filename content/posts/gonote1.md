## go语言的基础结构

go的基础组成有以下几个部分

+ 包声明
  必须在源文件非注释的第一行表明该文件属于哪个包，每个go程序都包含一个名为main的包，另

  + 文件名与包名没有直接关系，不需要将其定成同一个
  + 文件夹名与包名也没有直接关系
  + 同一个文件夹下的文件只能有一个包名
+ 引入包
  标识符以大写字母开头时，使用这种标识符的对象就可以被外部包的代码使用（只要客户端程序预先导入），这被称为导出，小写字母开头则对包外不可见，只在整个包里可见且可用
+ 函数
  main函数是每一个可执行程序所必须包含的，一般来说都是在启动后第一个执行的函数
+ 变量
+ 语句 & 表达式
+ 注释

*注：`{`不能放在单独一行*

## go语言基础语法

程序一般由关键字，常量，变量，运算符，类型和函数组成

可能使用到大中小三种括号

可能使用到 `,` `.` `;` `:` `...`

### go标记

go程序可以由多个标记组成，可以是关键字，标识符，常量，字符串，符号等

### 行分隔符

在go程序中，一行代表一个语句结束，不以分号为结尾，如果想将多个语句写在同一行，那就必须用分号区分，但在实际开发中此行为并不被鼓励。

### 注释

每一个包都应该有相关注释，单行注释使用 `//`，多行注释使用 `/*`和 `*/`不可以嵌套使用，一般用于包的文档描述或注释成块的代码

### 标识符

标识符用来命名变量，类型，由字母，数字，下划线组成，第一个字符必须是字母或下划线

### 字符串连接

通过 `+` 来实现

```go
package main

import "fmt"

func main() {
    fmt.Println("Google" + "Runoob")
}
```

### 格式化字符串

使用 `fmt.Sprintf`格式化字符串并赋值给新串

```go
package main

import (
    "fmt"
)

func main() {
    // %d 表示整型数字，%s 表示字符串
    var stockcode=123
    var enddate="2020-12-31"
    var url="Code=%d&endDate=%s"
    var target_url=fmt.Sprintf(url,stockcode,enddate)
    fmt.Println(target_url)
}
```

### 标识符可见性

大写其他包就能访问，小写就是私有的

若某函数公有，其返回值为某私有类型，函数是能正常调用且运行，但必须适用`:=`这样的操作符来自动识别变量类型

这样在和其他人进行函数方法通信时，只需约定好接口就行，内部实现使用方是看不到的，隐藏了实现

若在某个导出的结构体类型里，有一个未导出的字段那么该字段在其他包里无法初始化。

此外，一个导出的类型，包含了一个未导出的方法也一样，无法进行访问

若导出类型和未导出的类型相互嵌入，那么

若外部类型被导出而其内部某类型未导出，那么就不能使用字面值直接初始化它，只能先定义外部类型变量后，就可以对里面内部类型被导出的字段用外部类型访问初始化，因为当内部类型被嵌入后，其字段被提升到了外部类型中，这样它就可以被访问

若还想用`:=`那么字面值初始化时什么都不要做，因为内部类型未导出，所以不能使用字面值直接初始化内部类型里的字段，此外，内部类型未导出也意味着不能通过外部类型访问内部类型

总之，**`.`操作符前面的部分导出了，`.`操作符后面的部分才有可能被访问；如果`.`前面的部分都没有导出，那么即使`.`后面的部分是导出的，也无法访问。**

## 变量

变量名规则与其他语言一样，声明变量用var关键字

```go
package main
import "fmt"
func main() {
    var a string = "Runoob"
    fmt.Println(a)

    var b, c int = 1, 2
    fmt.Println(b, c)
}
```

### 变量声明

#### 无初始化指定变量类型

```go
var v_name v_type
v_name = value
```

这种情况时变量会默认为0值

各项数据类型的零值：

+ 数值：0
+ 布尔：false
+ 字符串：“”（空字符串）
+ 指针，数组，图，函数，接口等：nil

#### 根据值自行判断变量类型

```go
var v_name = value
```

#### 已经声明过的变量不能用:=

会有编译错误
:=实际上完成的是声明和赋值两个操作
`intVal := 1`实际上就是

```go
var intVal int 
intVal =1 
```

#### 多变量声明

```go
//类型相同多个变量, 非全局变量
var vname1, vname2, vname3 type
vname1, vname2, vname3 = v1, v2, v3


// 这种因式分解关键字的写法一般用于声明全局变量
var (
    vname1 v_type1
    vname2 v_type2
)
```

### 值类型和引用类型

所有基本类型都属于值类型，使用这些变量直接指向内存中的值
赋值时实际上是将一个变量的值进行拷贝赋给另一个变量
可以用 `&i`获取变量内存地址，值类型变量值放在堆中

更复杂的数据一般使用引用类型保存，一个引用类型的变量存储的是其值所在的内存地址，或内存中第一个字所在的位置
这个内存地址被成为指针，实际上也存在另外的一个值里
同一个引用指针指向的多个字可以是在连续的内存地址中（也是最快的存储方式），也可以将这些字分散存放在内存，每个字指示了下一个字所在的内存地址

### :=

使用变量的首选形式，但只能被用于函数体内而无法在全局变量中使用
*注：定义了某个局部变量却没在作用域里使用也会编译错误*
*但全局变量允许声明而不使用*
当然也能这样用

```go
a, _, c := 5, 7, "abc"
```

这叫并行或同时赋值，_可以用来抛弃值，作为一个只写变量但不得到其值，因为go里必须使用所有声明的变量，但有时并不需要使用从一个函数里得到的所有返回值
如果想交换变量，可以

```go
a, b = b, a;
```

并行赋值也可用于一个函数返回多个值时

```go
val, err = Func1(var1);
```

### 常量

一个简单值的标识符，运行时不会被修改的量，只可以是布尔型，数字型和字符串型

常量格式：

```go
const identifier [type] = value
```

可以省略说明符 `[type]`，因为编译器能根据变量值来判断类型，多个相同类型的声明也能简写为

```go
const c_name1, c_name2 = value1, value2
```

常量也能定义为枚举

```go
const (
    Unknown = 0
    Female = 1
    Male = 2
)
```

常量可以用 `len()`,`cap()`,`unsafe.Sizeof()`来计算函数表达式的值，但函数必须是内置函数，否则无法编译

#### iota

特殊常量，可以被编译器修改
在const关键字出现时将充值为0，每新增一行常量声明就将其计数一次（可以理解为const语句块中的行索引）
iota可以用作枚举值

```go
const (
    a = iota
    b = iota
    c = iota
)
```

上面的第一个iota等于0，在新的一行被使用前值都自动加一，所以

```go
const (
    a = iota  //a=0
    b         //b=1
    c         //c=2
)
```

iota在块中的增加与这一行表达式无关，即就算这一行是一个独立的赋值，只要之前有出现过iota，就能自增1

另，iota也带来不少有意思的操作

```go
const (
    i=1<<iota       //i=1，1左移0位仍为1
    j=3<<iota       //j=6，3左移0位为6
    k               //k=12，3左移2位为12
    l               //l=24，3左移3位为24
)
```

iota在同一个conse常量组里递增，当有新的const关键字时将重新计时
